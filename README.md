# express-ts-template

This is a simple template with Express and TypeScript. The objective of this template is to create REST microservices easily and quickly.

## Tech Stack

**Server:** Node, Express

**JWT**: For authentication

**Swagger**: For API documentation

**MVC**: This project is inspired by the MVC design pattern.

## Run Locally

Clone the project:

```bash
  git clone https://github.com/abelgonzalezr/express-ts-template
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn build && yarn dev
```


## Running Tests

To run tests, run the following command

```bash
  yarn test
```


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`SECRET_KEY` for Jwt

`PORT`
